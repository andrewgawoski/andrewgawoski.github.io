Perdix

Perdix was prototyped and developed in 2016 as a stylish 2D action-platformer similar to games like “Ori and the Blind Forest” and “Muramasa the Demon Blade.” Players battle their way through the world with the aid of a wind deity as they attempt to return the sun to the sky before the break of day.

It was developed independently in C# with the Unity engine for gamepad/console play. Development was halted due to scope. I’ve included an early alpha build, design documentation, and several code samples.

Luna Limit

Luna Limit was developed independently with Unity. It is a single player tile-based game with a strong simple math influence. Players activate tiles to change surrounding moon phases in a way similar to adding simple numbers. Players attempt to create as many full moons as possible by flipping tiles along the edge of the board.

In the end, adding phases of the moon together proved not to be intuitive enough. Were I to revisit this title, I’d consider approaching this from the perspective of an eclipse, as it seems easier for people to visualize a circle moving across the sun.

Summoner Timers

Summoner Timers is an assistant application for players of Riot Games’ League of Legends. The application allows players to keep track of the cooldown for various different in game abilities not displayed by the game’s UI. Users may select from automatic and manual ability entry.

Uses Riot Games’ RESTful API to pull live game data for the purpose of automatically populating the application’s timers. Summoner Timers is a native android application developed in Java independently as an academic project at UCSC.
